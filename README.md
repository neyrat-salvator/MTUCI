# MTUCI
Репозиторий для учебы в институте

27.11.2025
- Готова структура для выполнения задач по Распределенным операционным системам
- Для Элпит не готово (НЕ РАБОТАЕТ)
- Для КТП не готово (НЕ РАБОТАЕТ)
- В алгоритмах используется конвертирование разных типов данных. Байты конвертируются между собой не округленным значением "1000", а действительным "1024"

РОС.
1. Запустить main.py из корневой директории

python main.py

2. Вписать:
- ФИО полностью на русском языке
- Группу полностью
- Номер зачетной книжки полностью
- Текущий номер по списку

3. Скопировать из терминала нужные значения и вставить в отчет
